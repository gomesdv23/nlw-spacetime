## 🖥️ Projeto
Esse é um projeto web responsivo de uma capsula do tempo para exibir memórias de uma linha do tempo.
## 🚀 Tecnologias
Esse projeto foi desenvolvido no NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
